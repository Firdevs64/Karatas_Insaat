# Karataş İnşaat – Kurumsal Web Sitesi

Karataş İnşaat firması için geliştirilmiş, modern ve responsive bir kurumsal web sitesidir.
Proje, firmanın hizmetlerini ve projelerini tanıtmak amacıyla hazırlanmıştır.

---

## Kullanılan Teknolojiler

- React
- TypeScript
- Vite
- Mantine UI
- React Router

---

## Proje Yapısı

- Hero alanı (tam ekran görsel + başlık)
- Hizmetler bölümü
- Projeler vitrin bölümü (kart yapısı)
- Proje detayları (görsel ve video içerik)
- İletişime yönlendiren CTA alanı

---

## Özellikler

- Responsive tasarım (mobil / tablet / masaüstü)
- Bileşen tabanlı React mimarisi
- Statik görsel ve video kullanımı
- Temiz ve sade arayüz
- Kurumsal web sitesi yapısına uygun tasarım

---
